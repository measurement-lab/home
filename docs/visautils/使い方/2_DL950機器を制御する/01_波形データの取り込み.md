## 1_oscilloクラスインスタンスの初期設定まで

- **mesDeice**パッケージの**oscillo**クラスの初期設定までのPythonスクリプトは，下記のようになります．

```python
from visautils import mesDevice, visaDL950

ndata   = 4096
freq    = 50
vch     = (1,1)
os_tch  = "EXT"
os_clch = "EXT"
average = 20

DL950 = visaDL950.visaDL950("ENV_DL950_RESNAME")
DL950.open()

oscillo = mesDevice.oscillo(freq, ndata, os_tch, os_clch, average=average)
chs = [vch]
oscillo.initial_setting(DL950, chs)
```

- 最初に，DL950クラスインスタンスを生成し，open()関数でDL950機器と接続します．
- 次に，波形データを取り込む際の周波数などの初期設定項目を引数として，mesDeviceモジュールのoscilloクラスを生成します．
- 最後に，DL950クラスインスタンスと，取込む波形データのチャネル群のリスト（装着モジュールのチャネルからトリガ信号を取り込むケースでは，トリガ信号を取り込むチャネルも含めて下さい）を引数として，oscilloクラスインスタンスのinitial_setting()関数を実行することで，DL950機器を制御するためのoscilloクラスインスタンスの初期設定が完了しました．

### 1.1_トリガ信号およびクロック信号

- DL950機器で波形データを取り込むには，波形データの電圧信号の他に，**トリガ信号**と**クロック信号**が必要になります．
- DL950機器では，本体内部で，このトリガ信号もしくはクロック信号を用意する（検出する）機能もあります．さらに，外部トリガ信号には，**外部トリガ入力端子**と，装着モジュールのチャネルの二種類の端子から取り込むことができます．
- いずれのケースにおいても，visautilsパッケージは，oscilloクラスインスタンスを生成する際の**os_tch**引数と**os_clch**引数に値を指定することで対応できます．
- トリガ信号とクロック信号の，それぞれの接続状況に対する値は，下記のようになります．

【トリガ信号】

| DL950機器の接続端子 | 外部トリガ入力端子 | 装着モジュールのチャネル |
| ------------------- | -------------------- | ---------------- |
| **os_tch**引数で設定する値                    |   "EXT"                   | (1,1)～(x,2)<br>（装着モジュールのスロット番号による）                 |

【クロック信号】

| DL950機器の接続端子 | 外部クロック入力端子 | 内部クロック使用 |
| ------------------- | -------------------- | ---------------- |
| **os_clch**引数で設定する値                    | "EXT"                     | "INT"                 |


## 2_電圧波形データの取り込み

- **chs**引数に取り込む電圧波形データのチャネル番号のリストを設定して，**capture_waves**()関数を実行することで，該当するチャネル番号の波形データのリストが戻り値として取得できます．

- (1,1)チャネルから電圧波形データを取り込むPythonスクリプトを，下記に示します．
- capture_waves()関数の戻り値は，取り込んだ波形データ（numpy配列）のリストとなっています．
- 下記の例では，取り込んだ波形データのチャネル数は1なので，戻り値のリストのインデックス0番目が，取り込み波形データそのものとなります．

```python
chs = [(1,1)]
vss = oscillo.capture_waves(chs)
vs_1_1ch = vss[0] # (1,1)チャネルの取り込んだ波形データ
```

- 複数チャネルから波形データを取り込むケースも，capture_waves()関数に，チャネル番号のリストを設定すれば良いです．
- この場合，戻り値は，capture_waves()関数に設定した，取り込みチャネル番号の順に，取り込み波形データが収納されたリストとなります．

```python
chs = [(2,1), (1,1), (1,2)]
vss = oscillo.capture_waves(chs)
vs_2_1ch = vss[0] # (2,1)チャネルの取り込んだ波形データ
vs_1_1ch = vss[1] # (1,1)チャネルの取り込んだ波形データ
vs_1_2ch = vss[2] # (1,2)チャネルの取り込んだ波形データ
```
