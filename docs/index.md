# About 

- 測定機器をプログラムで自動制御する測定システムを開発しています．
- 測定機器には，**SCPI**と呼ばれる規格に沿った制御用のコマンドがメーカから提供されています．
- SCPIコマンドを測定機器に送受信するには，各種プログラミング言語（C#, VB, MATLABなど）を使うことができますが，**Python**を使います．

## visautilsパッケージ

- **visautils**パッケージは，測定機器をPythonを使って制御するためのドライバ（測定機器を制御する目的毎に，複数のSCIPコマンドを組み合わせたライブラリ）です．
- SCIPコマンドを個別に送受信するのに比べて，格段に効率良く測定システムを作ることができます．
- 現在は，NF回路設計ブロック社のWF1968ファンクション・ジェネレータと，横河計測社のDL950スコープコーダを制御する機能を提供します．
- visautilsパッケージの使い方を，上の**visautilsパッケージ**タブで紹介しています．visautilsパッケージタブをクリックしてみて下さい．

## Python関連

- visautilsパッケージを使って測定機器を制御するには，Pythonに関する知識も必要となります．
- Python関連の情報を，上の**Python関連**タブで紹介しています．かなり個人的見解に基づいたものですが，WindowsマシンにPython環境を構築（インストール）する方法や，仮想環境，pipコマンドとcondaコマンドの違いなどを紹介しています．
