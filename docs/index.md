# About 

- 測定機器をプログラムで自動制御する測定システムを開発します．
- 測定機器は，**SCPI**と呼ばれる制御用のコマンドがメーカから提供されています．
- SCPIコマンドを測定機器に送受信するには，各種プログラミング言語（C#, VB, MATLABなど）を使うことができますが，**Python**を使います．

## visautilsパッケージ

- **visautils**パッケージは，測定機器をPythonスクリプトを使って制御するためのドライバです．
- SCIPコマンドを直接送受信するのに比べて，格段に効率良く測定システムを作ることができます．
- 現在は，NF回路設計社のWF1968ファンクション・ジェネレータと，横河計測社のDL950スコープコーダを制御する機能を提供します．
